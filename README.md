# MVC2-Homepage-Business-card-
MVC2 패턴의 CRUD 명함 관리 홈페이지


## 팀원
  - 송민혁
  - 김지훈

## 개발 언어, 환경, 프레임워크
   - Jsp & Servlet
   - MySQL
   - Apache Tomcat
   - Eclipse
   - HTHL, CSS, JS
   
   
## 프로그램 구현 기능

    1. 사용자의 정보는 아이디, 비밀번호, 이메일, 전화번호가 반드시 있어야 한다.
    2. 명함에는 명함 이름이 반드시 있어야 한다.
    3. 명함에는 회사 이름, 명함 이름, 전화번호, 부서, 직책, 이메일이 들어 갈 수도 있다.
    4. 회사 정보는 회사명에 의해 따로 관리된다.
    5. 사용자는 명함 정보와 회사 정보를 수정할 수 있다.
    (명함의 경우 사용자 아이디 정보를 확인하여 목록 확인 수정 가능)
    6. 사용자는 본인이 입력한 명함의 정보만 확인할 수 있다.
    7. 사용자는 다른 사람이 입력한 회사 정보를 확인할 수 있다.
    (회사 정보의 경우 별도의 아이디 정보 확인이 없음)
